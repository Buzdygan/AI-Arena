may_contest_views¶

ai_arena.contests.may_contest_views.match_details(request, *args, **kwargs)¶

Displays details of a Match object with given match_id.

ai_arena.contests.may_contest_views.may_contest_send_bot(request, *args, **kwargs)¶

This view is used when user wants to send bot from game details view. It is different from simple send bot view, because game is known, so there is no need to pick it form list.

ai_arena.contests.may_contest_views.my_results(request, *args, **kwargs)¶

Selects from database results of all matches against a User’s Bot.

Contains both ranked and test matches.

ai_arena.contests.may_contest_views.online_bot_creation(request, *args, **kwargs)¶

Page where everyone can prepare bot online and submit it to contest.

ai_arena.contests.may_contest_views.show_ladder(request)¶

Generates, refreshes and displays current ranking of may Contest.

ai_arena.contests.may_contest_views.testing(request, *args, **kwargs)¶

View responsible for displaying form to upload Bots to test.

Using provided form User can upload one or two Bot’s source codes. In case of uploading two source codes system performs a Match between these two Bots. If User uploads only one source code, then system performs Match against default Bot.
