class ai_arena.contests.models.Bot(*args, **kwargs)¶

Stores info about single Bot. Bot is owned by User and connected with Game. Bot has its own BotContestRanking where the info about Bot’s matches is accumulated.

compile_bot()¶

Compile source file to directory with source file

class ai_arena.contests.models.BotRanking(*args, **kwargs)¶

class ai_arena.contests.models.Contest(*args, **kwargs)¶

There are Matches played within the Contest and they results sum up to the ContestRanking.

generate_group_ranking()¶

Method that generates rank list for the contest. It checks whether all matches have been played. If not, it launches the missing ones by gearman.
