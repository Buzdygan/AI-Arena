game_views¶

ai_arena.contests.game_views.create_new_game(request, *args, **kwargs)¶

Prepares a form to render for creating new game. Later it checks results, and if everything is OK it saves new Game object to database

ai_arena.contests.game_views.delete_game(request, *args, **kwargs)¶

Deletes game which id is equal to game_id.

The view performs a safety check - it makes sure that user has apprioprate previllages (is either a moderator of this game or admin).

ai_arena.contests.game_views.edit_game(request, *args, **kwargs)¶

Allows user to edit game description and other details. Takes one extra argument - game_id - describing id of a game to edit.

The view takes care of safety issues - it checks if a user calling this view has apprioprate privillages (is a moderator of the game or staff member)

ai_arena.contests.game_views.game_details(request, game_id, error_msg=None)¶

Displays detailed information about game with id equal to game_id If game_id is not given or there is no Game object with id equal to game_id then Exception is thrown

ai_arena.contests.game_views.game_list(request)¶

Displays list of available games

ai_arena.contests.game_views.parse_game_details(game)¶

Helper function preparing subobjects of game object to display.

ai_arena.contests.game_views.parse_line(line, lang)¶

Parses line of source code returning list of pairs (class, value), where class is to be used in html code to apply correct css

ai_arena.contests.game_views.show_source(request, game_id)¶

Displays source code of the judge connected with game with id=game_id
