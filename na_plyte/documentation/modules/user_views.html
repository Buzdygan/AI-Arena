user_views¶

ai_arena.contests.user_views.edit_profile(request, *args, **kwargs)¶

Called when user wants to edit his profile page. Note that none of the fields in returned form is required.

ai_arena.contests.user_views.getDefaultUserProfile(user)¶

Helper function that creates and saves default user profile. Used when user doesn’t have his profile yet.

ai_arena.contests.user_views.show_contests(request, *args, **kwargs)¶

displays list of contests in which user is participating.

ai_arena.contests.user_views.show_news(request, *args, **kwargs)¶

Displays a list of news linked with the user. News are created in variety of situations: when creating a new game, sending a bot for a game, editing a profile and so on.

ai_arena.contests.user_views.show_profile(request, *args, **kwargs)¶

Displays user’s profile. If user calls this function for the first time (or user profile doesn’t exist of any reason) it also creates new profile with default settings (which can be customized later).
