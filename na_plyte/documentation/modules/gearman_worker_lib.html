gearman_worker¶

ai_arena.gearman_worker_lib.single_match(gearman_worker, gearman_job)¶

Gearman worker main function.

It’s responsible for getting argunemts from Gearman server, parsing them and finally calling Nadzorca’s play function (which performs Match between given Bots).

Then it stores results in match object (received from Gearman server) and saves changes to database.
